# LMS Components Examples

This directory contains comprehensive examples demonstrating how to use the LMS Components library.

## Running the Examples
```bash
cd examples
npm install
npm start
```

The examples app will open in your browser at `http://localhost:3000`.

## Examples Included

### Basic Question Components
- Multiple Choice (single and multi-select)
- True/False (with different display modes)
- Short Answer (with validation)
- Essay (with word/character counting)
- Fill in the Blank
- Matching

### Quiz Components
- Simple quiz with question-level submission
- Timed quiz
- Quiz requiring all answers
- Linear quiz (no navigation back)

### Advanced Examples
- Custom question renderers
- Quiz with review mode
- Event callbacks and analytics

## Usage Patterns

Each example demonstrates:
- Component configuration
- State management
- Event handling
- Validation
- Feedback display
- Customization options
